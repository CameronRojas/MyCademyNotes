HTML tags

<div></div> the div tag helps divide the body of an html document into smaller sections for easier styling and manipulation

<h1></h1> the h1-h6 tags are used for headings, most browsers have default styles

<span></span> for short pieces of text, usually on the same line

<em></em> italicizes text (i.e. ITALICS)

<strong></strong> embolden text (i.e. BOLD)

<br> goes to next line
(i.e. we are on next line)

<ul></ul> creates unordered list

  • <li></li> adds elements to the list

<ol></ol> creates ordered list

  1. <li></li> adds elements to the list

<img> allows the addition of images, required attribute src=""

HTML attributes

id="" specifies different content, the highest order of specificity

src="' is equal to the URL of the image

alt="' is alternative text that describes the image/video, helps with SEO and more user friendly
